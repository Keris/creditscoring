Credit Scoring Toolkit
===================================================
credit-scoring-toolkit is a credit scoring toolkit with Python that provides functions for mathematical modeling of credit risk.
You can model probabilities of default, create credit scorecards, perform credit portfolio analysis, and backtest models to assess potential for financial loss.

.. Note::

   This package is still under planning.
